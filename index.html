MY PROJECTS

FITS IN BICYCLE

Designed and developed a bicycle-mounted health monitoring system using ESP32 to measure heart rate, SpO₂, and body temperature. Integrated MAX30102 pulse oximeter and temperature sensor with real-time data display on an LCD. Implemented I²C communication and sensor data processing for reliable physiological monitoring. The system provides on-bike health insights for rider safety and fitness awareness.

ESP32 I²C communication Hardware Interfacing

MINI RADAR SYSTEM

Designed and developed a radar detection system using Arduino, ultrasonic sensor, and servo motor for object detection. Implemented real-time distance measurement by sweeping the sensor using a motor mechanism. Visualized detected objects using Processing software with a radar-style graphical interface. The project demonstrates embedded system control, sensor interfacing, and data visualization.

Arduino Processing Data Visualization
🤖

AUTOMATED LINE CROSSING DETECTION SYSTEM

The Automated Line Crossing Detection System addresses the problem of unauthorized or accidental boundary crossing in restricted or monitored areas such as sports grounds, airport queues, and controlled zones. The system uses an Arduino microcontroller and a GP2Y0E03 distance sensor to continuously monitor a defined boundary and detect crossing events. When a crossing is detected, the system triggers an alert, enabling improved security, safety, and orderly movement in controlled environments.

Arduino Sensor Interfacing Embedded C
